### Process Flow
1. **Prepare:** Use this CSV as source of truth
2. **Research:** For each book, gather:
   - High-res cover image
   - 250-400 word description (why banned, cultural impact)
   - Specific ban incidents (dates, locations, organizations)
   - Award/recognition details (for credibility)
   - Goodreads link + reader reviews excerpts (optional)
3. **Generate:** Create markdown files using existing template (or batch creator)
4. **Test:** Deploy to staging, verify all links work
5. **Commit:** Single batch commit with all 40-50 books
6. **Deploy:** Cloudflare Pages
