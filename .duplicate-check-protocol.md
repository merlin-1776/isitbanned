# IsItBanned Duplicate Detection Protocol

## Purpose
Prevent duplicate book entries before they're committed and pushed to the repository. Duplicates waste storage, confuse search rankings, and split SEO authority.

## Detection Method (Execute Before Adding Any Books)

### Step 1: Build Current Inventory Index
```bash
# Extract all existing books into a dedupe-safe index
python3 ./scripts/build-inventory-index.py
# Output: .inventory-index.json (cached)
```

### Step 2: Pre-Flight Check (For Each New Book)
Check the candidate against **three dimensions**:

#### 2A. ISBN Match (Highest Priority)
```
If book_isbn exists in inventory:
  - BLOCK immediately (same book, different title/slug likely)
  - Flag for merge decision
  - Do NOT create new file
```

#### 2B. Title+Author Match (Normalized)
```
Normalize: lowercase, remove punctuation, remove "the/a/an" prefix
If normalized_pair exists in inventory:
  - Check if it's a legitimate variant:
    - Volume numbers (vol. 1, vol. 2) → OK to create separately
    - Edition differences (expanded, revised, graphic) → OK if ISBN differs
    - Series entries → OK if unique ISBN
  - If no ISBN difference: FLAG as duplicate
```

#### 2C. Fuzzy Title Match (Levenshtein Distance)
```
If similar_ratio(candidate_title, existing_title) > 0.85:
  - Manual review required
  - Check author match
  - Check ISBN
  - Decide: same book or different?
```

### Step 3: Report Before Commit
```
Generate dedup report:
  ✅ Clean (no issues)
  ⚠️  Variant (legitimate edition/volume, OK to add)
  ❌ Duplicate (block and suggest merge/redirect)
```

## Implementation Script

### `scripts/check-duplicates.py` (To Use Before Adding Books)
```python
#!/usr/bin/env python3
import sys
import json
import re
from difflib import SequenceMatcher

def normalize_title(title):
    """Remove articles, punctuation, lowercase"""
    t = title.lower().strip()
    t = re.sub(r'^(the|a|an)\s+', '', t)
    t = re.sub(r'[^\w\s]', '', t)
    return ' '.join(t.split())

def fuzzy_match(s1, s2, threshold=0.85):
    """Levenshtein-style similarity"""
    ratio = SequenceMatcher(None, s1, s2).ratio()
    return ratio >= threshold

def check_duplicates(candidates_dict, existing_inventory):
    """
    Args:
        candidates_dict: {slug: {title, author, isbn}}
        existing_inventory: {slug: {title, author, isbn}}
    
    Returns:
        {slug: "✅ CLEAN" | "⚠️  VARIANT" | "❌ DUPLICATE"}
    """
    results = {}
    isbn_index = {book['isbn']: slug for slug, book in existing_inventory.items() if book.get('isbn')}
    
    for c_slug, c_data in candidates_dict.items():
        c_isbn = c_data.get('isbn')
        c_title = c_data.get('title', '')
        c_author = c_data.get('author', '')
        
        # ISBN check (highest priority)
        if c_isbn and c_isbn in isbn_index:
            existing_slug = isbn_index[c_isbn]
            existing_title = existing_inventory[existing_slug].get('title', '')
            results[c_slug] = f"❌ DUPLICATE (ISBN {c_isbn} exists as '{existing_slug}')"
            continue
        
        # Normalized title+author check
        c_norm = normalize_title(c_title)
        found_match = False
        for e_slug, e_data in existing_inventory.items():
            e_norm = normalize_title(e_data.get('title', ''))
            e_author = e_data.get('author', '').lower().strip()
            c_author_norm = c_author.lower().strip()
            
            if c_norm == e_norm and c_author_norm == e_author:
                # Same normalized title and author - likely duplicate
                # But check if it's a legitimate variant (vol, edition, etc)
                if any(x in c_data.get('title', '').lower() for x in ['vol', 'volume', 'edition', 'graphic', 'revised']):
                    results[c_slug] = f"⚠️  VARIANT (similar to '{e_slug}', but appears to be edition/volume)"
                else:
                    results[c_slug] = f"❌ DUPLICATE (title/author match: '{e_slug}')"
                found_match = True
                break
        
        if found_match:
            continue
        
        # Fuzzy match as last resort
        for e_slug, e_data in existing_inventory.items():
            e_title = e_data.get('title', '')
            e_author = e_data.get('author', '').lower().strip()
            
            if fuzzy_match(c_norm, normalize_title(e_title)) and c_author_norm == e_author:
                results[c_slug] = f"⚠️  FUZZY MATCH ('{e_slug}': {int(SequenceMatcher(None, c_norm, normalize_title(e_title)).ratio() * 100)}% similar)"
                found_match = True
                break
        
        if not found_match:
            results[c_slug] = "✅ CLEAN"
    
    return results
```

## Workflow: Before Adding Any New Books

1. **Prepare candidates** as: `{slug: {title, author, isbn}, ...}`
2. **Run check**:
   ```bash
   python3 scripts/check-duplicates.py --candidates new_batch.json
   ```
3. **Review output**:
   - ✅ CLEAN → safe to add
   - ⚠️  VARIANT → decide: merge descriptions or keep separate?
   - ❌ DUPLICATE → remove from batch, update existing entry instead
4. **Commit only clean batch**
5. **Document any merges/redirects** in changelog
